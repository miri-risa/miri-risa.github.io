 README

This app loads kanji questions from a questions.json file.

If a questions.json file exists in the same folder as this HTML file, it will automatically load on startup.

You can also manually upload another JSON file at any time.
JSON Template

[
  {
    "kanji": "水",
    "answers": ["mizu", "みず"],
    "meaning": "water"
  },
  {
    "kanji": "火",
    "answers": ["hi", "ひ"],
    "meaning": "fire"
  }
]
